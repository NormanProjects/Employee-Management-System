# Functional Requirements

## User Management
- The system will allow an admin to create, update, and deactivate employees.
- The system will assign roles to employees (ADMIN, HR, EMPLOYEE).

## Employee Management
- The system will store employee personal details.
- The system will allow updating employee information.

## Authentication
- The system will allow users to log in using username and password.
- The system will restrict access based on user roles.

## Leave Management (Optional)
- Employees can apply for leave.
- Managers can approve or reject leave requests.
